# Skills Hub

Skills Hub is an Android-based mobile application built using Java and XML in Android Studio.  
It is designed for students, freelancers, and creators who want to learn new skills, showcase their talents, and earn money through freelance work.  
The main goal of this app is to create a single platform where users can both learn and earn by connecting with other skilled individuals and clients.

## Features

### Learn Section
- The Learn section provides curated learning resources, mainly YouTube videos, categorized under different skills like:
  - Graphic Design
  - Video Editing
  - Content Writing
  - Web Development
  - UI/UX Design
- Each skill has recommended free tools and learning paths.
- Users can track their learning progress.
- There is also a search bar to easily find courses or skills.
- Both free and paid course links are available.

### Freelance Section
- Freelancers can list their services and connect with clients.
- They can specify their specialization, set project prices, and manage work through the app.
- A simple wallet system is included for handling payments.
- There is a basic job-matching system to help users find suitable projects.
- Users can build and manage their portfolio to showcase previous work.

### Other Features
- Secure login and signup using Firebase Authentication.
- Personalized dashboard after login.
- Refer and Earn program available on the homepage.
- Profile page with skill tags, earned badges, and completed project history.
- A clean and simple user interface focused on easy navigation and usability.


## Future Plans
- AI-powered skill suggestions based on user activity.
- Automatic freelance match recommendations.
- Analytics dashboard for users to track learning and earnings.
- Improved chat system and notification support.


## App Flow
1. Login/Signup page connected with Firebase.
2. Main page with two carousels showing popular skills and featured freelancers.
3. Navigation panel that links to Learn, Freelance, Chat, and Profile sections.
4. Skills section where users can browse, search, and access free or paid courses.
5. Freelance work page where users can post or apply for projects.
6. Chat page for communication between freelancers and clients.


## Tech Stack
- **Frontend:** XML (Android Layouts)
- **Backend:** Java
- **Database:** Firebase Realtime Database / Firestore
- **Authentication:** Firebase Auth
- **IDE:** Android Studio
- **Build System:** Gradle


## Setup Instructions

1. Clone the repository  
   `git clone https://github.com/devanshgupta-git/skillshub.git`

2. Open the project in Android Studio.

3. Connect Firebase:  
   Go to Tools → Firebase → Connect your app with Firebase for Authentication and Database.

4. Sync Gradle and run the project on your emulator or Android device.
